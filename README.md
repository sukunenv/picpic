## ℹ️ About
Ini adalah versi pertama (v1.0) aplikasi web Picpic Cafe yang sudah berjalan di production.
Dibangun menggunakan Vue.js + Laravel dan di-deploy di Railway.

## ✨ Features
- 📋 Menu digital Picpic Cafe
- 🛒 Order system
- 👨💼 Admin dashboard
- 📊 Laporan & analytics
- 🖨️ Cetak struk thermal printer

## 🛠️ Tech Stack
- Frontend: Vue.js 3
- Backend: Laravel
- Database: MySQL
- Deployment: Railway
- Domain: kedaipicpic.com
